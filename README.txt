== Description ==

This is an all types AdBlock detector plugin for WordPress websites. Start saving your ads money by informing users to whitelist your website.

Features

- All the code is optimized, minified, analyzed with lighthouse & GTmetrix.
- No jQuery used only native Javascript.
- Customizable title, description, and animations.
- Supported by large collections of famous and custom themes.
- Tested on +50 chrome, firefox, browsers with AdBlock extensions.
- Simple design and UI.
- Disabling access to dev tools and context menu (Mac & Windows).

How to install?

- Unzip the downloaded plugin file.
- Move the folder to your WordPress installation "/wp-content/plugins/".
- Activate the plugin from your WordPress admin interface clicking on "Activate".
- Test the plugin working AdBlock extensions or browsers.
